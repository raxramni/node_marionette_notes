# Boilerplate npm Webpack Backbone and Express

## Install

1. ```npm install```
2. ```npm start```
3. ```http://localhost:3000```

## Dev

1. ```npm run dev```
2. ```http://localhost:3000```

## Test & cover

* ```npm test```
* ```npm run cover```

## Build

* ```npm run build```

## Recomendations

Use Node >= 4
